# Hasil Evaluasi Model

Folder ini menyimpan hasil kuantitatif dari model Machine Learning (Random Forest) yang digunakan.

## Isi Folder
- File `.txt` atau grafik yang berisi Confusion Matrix.
- Metrik evaluasi performa model: Accuracy, Precision, Recall, F1-Score (APRF).
- Ringkasan luas perubahan lahan & kekeruhan air, beserta grafik/tabel hasil perhitungan.
