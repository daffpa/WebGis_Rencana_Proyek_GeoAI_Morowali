# Google Earth Engine Scripts

Folder ini digunakan untuk menyimpan script JavaScript dari Google Earth Engine yang digunakan dalam proses analisis data.

## Yang harus dimasukkan ke sini:
- Script Preprocessing (Masking awan, komposit)
- Script Ground Truth sampling
- Script training model Random Forest & split dataset
- Script klasifikasi & deteksi perubahan
- Script vectorization & ekspor GeoJSON
